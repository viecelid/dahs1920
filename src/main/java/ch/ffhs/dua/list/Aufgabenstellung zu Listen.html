Aufgabe zu verketteten Listen

Implementieren Sie eine doppelt gekettete Liste sowie eine dazugehörige Iterator-Klasse. Es dürfen kleine Klasse aus dem Package java.util verwendet werden.

Die Listenklasse soll die folgenden Methoden aus dem Interface java.util.List implementieren:

  • add(E)
  • add(int, E)
  • get()
  • set(int, E)
  • remove(int)
  • size()
  • iterator()
Die Iterator-Klasse sollte möglichst effizient implementiert sein und die folgenden drei Methoden implementieren:
  • hasNext()
  • next()
  • remove()

Verwenden Sie die Klasse LinkedList dieses Projektes und ersetzen Sie die TODOs.
